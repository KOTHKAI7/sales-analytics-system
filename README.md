# Sales Analytics System

## Overview
A Python-based system to clean, validate, and analyze sales transaction data.

## Features
- Handles messy pipe-delimited data
- Removes invalid records
- Cleans numeric and text formatting issues
- Generates summary sales analytics

## How to Run
1. Ensure Python 3 is installed
2. Place `sales_data.txt` inside the `data/` folder
3. Run the program:

```bash
python main.py
